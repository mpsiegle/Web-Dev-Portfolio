Off Square Brewing
Website Redesign

Objective

The purpose of this project was to heuristically evaluate a website and create a redesign that would have better functionality based on the results of the evaluation.

Process

Step one was heuristically evaluating the original website and finding any areas of user frustration. Next was brainstorming a redesign that could fix the discovered issues and create better user flow. Then I drafted low-fidelity wireframes that helped visualize the website redesign. Finally, the new website design was fully visualized in high fidelity via Figma.

Results

The final redesign successfully addressed all issues identified in the heuristic evaluation and was completed on time and under budget. This project focused on visual design only and was not fully coded or developed into a functional site. The full redesign is available on my Figma profile, and a mockup can be viewed on Behance.
