Blockhead Beerworks
Website Redesign

Objective

This project involved creating a functional website for a business. The website was to use HTML and CSS for styling and JavaScript for functionality. The brief indicated the website needed to have functioning JavaScript for e-commerce aspects (such as the product switcher) and the guessing game.

Process

The first step was heuristically evaluating the existing website to see where a new site could improve. Next was laying out a color scheme and website layout that would incorporate the required content. The team was then tasked with styling the HTML and CSS and then adding fundamental JavaScript. After incorporating all the code together, I did troubleshooting and retested for bugs before confirming I had accomplished all goals of the project brief. Finally was submitting the final deliverables to the client.

Results

The project wrapped up on time and on budget, delivering a clean, fully functional experience that reflects the intended design and user flow. Explore the completed build on my GitHub profile, and check out the high-fidelity mock-up on Behance.
